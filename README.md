## 🎯 Mission

Ce dépôt est une **bibliothèque vivante d'architectures d'entreprise**.

Je ne me contente pas de documenter des outils.  
Je documente **des décisions** : pourquoi cette technologie, dans ce contexte, avec ces contraintes, pour répondre à ce besoin métier.

Chaque architecture est une étude de cas qui répond à la question :

> *"Si je devais reconstruire cette infrastructure aujourd'hui, que ferais-je et pourquoi ?"*
